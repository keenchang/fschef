# fschef

## 一、安裝

安裝 Python3.6 以上的版本
使用以下命令安裝 Flask 和其他需要的套件：

```
pip install -r requirements.txt
```

## 二、開發技術

**前端：**

- HTML
- CSS
- JavaScript
- Bootstrap

**後端：**

- Python
- Flask

**部署：**

- Heroku

**版本控制：**

- Git
- GitHub

**資料庫：**

- PostgreSQL

**第三方金流：**

- 藍新金流
- Line Pay
